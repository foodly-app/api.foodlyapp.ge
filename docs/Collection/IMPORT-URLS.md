## 🚀 Quick URL Import

### Step 1: Import Collection
1. Open Postman
2. Click **Import** → **Link** tab
3. Paste this URL:
```
https://raw.githubusercontent.com/foodly-app/api.foodlyapp.ge/main/docs/Collection/FOODLY-API-Complete.postman_collection.json
```
4. Click **Continue** → **Import**
